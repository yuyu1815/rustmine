# Stevenarella Project Context

This glossary names project and operator vocabulary used to route Stevenarella
work. It is a language map only; evidence, decisions, and implementation facts
belong to their owning surfaces.

## Language

**Active protocol version**:
The protocol version selected by the active manifest or task route.
Use when a version-specific fact is needed. Not: a permanent global default.

**Helper trust surface**:
The boundary around what a helper script, subagent, or generated file can be
trusted to show. Use when reading helper output as evidence with a named scope.

**Living doc**:
A document that changes when durable understanding changes. Not: a ceremony
file updated only because work happened.

**Oracle artifact**:
A jar-backed or generated reference item used to answer a bounded compatibility
question. Not: a guess, convenience fixture, or broad playability claim.

**Orientation layer**:
The entrance map that shows where to read next. Not: the home for durable
claims or detailed facts.

**Owner shard**:
The smallest document or artifact surface responsible for a kind of knowledge.
Use when choosing where a durable update belongs.

**Phase lens**:
A client-load viewpoint used to locate which part of the playable route is under
discussion. Not: a replacement for the owning domain or evidence source.

**Recovery pointer**:
A short pointer that restores current location and next read after reset,
handoff, or compaction. Not: a durable evidence store.

**Reinterpretation loop**:
The repeated work caused when terms, ownership, or evidence boundaries are not
captured clearly enough for the next agent to reuse.

**Reset-prone checkout**:
The implementation checkout that may be replaced, reset, or regenerated.
Use when deciding what must live outside the code under test.

**Scoped PASS**:
A pass claim that names exactly what boundary was checked. Not: a broad
compatibility or playability claim.

**Witness**:
A secondary source, smoke run, probe, or comparison that explains or
cross-checks a claim. Not: the final authority when official sources exist.

**Yuzu**:
The read-only operator mirror and judgment lens for AI collaboration.
Use when checking visual routing, evidence posture, helper-output trust, or
responsibility boundaries. Not: a task workflow, glossary store, mutable project
state, or evidence ledger.
